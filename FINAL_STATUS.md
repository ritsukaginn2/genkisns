## 🚀 可立即使用

### 1. 启动后端

```bash
cd services/llm-proxy
cp .env.example .env

# 编辑 .env，添加 API keys:
# DEEPSEEK_API_KEY=sk_...
# DASHSCOPE_API_KEY=sk_...
# ARK_API_KEY=...

docker-compose up -d
```

### 2. 运行 iOS 应用

```bash
cd apps/mobile
flutter pub get
flutter run -d iPhone
# 或其他设备 ID
```

### 3. 用户操作流程

1. **首次打开**
   - 自动生成 Installation ID（保存到设备）
   - 初始化 LLM 客户端
   - 初始化 IAP（恢复之前的购买）

2. **发布笔记**
   - 输入文本和选择图片/视频
   - 点击"发布笔记"
   - App 调用 LLM 生成评论
   - 如果有额度，显示 AI 评论
   - 如果没额度，显示本地模板评论

3. **查看额度**
   - 点击"我的" → "权益和额度"
   - 看到剩余额度和重置时间
   - 如果是免费用户，可以点击"升级到 Pro"

4. **购买 Pro**
   - 点击"¥99/年 - 立即购买"
   - iOS 显示购买确认
   - App 将凭证发送给后端验证
   - 后端验证成功后，额度升级到 3000/天

5. **备份和导出**
   - 点击"我的" → "iCloud 同步" 自动备份
   - 点击"导出数据"下载 JSON
   - 支持多设备恢复
