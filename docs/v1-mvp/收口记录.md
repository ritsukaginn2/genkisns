# GenkiSNS V1 收口记录

> 日期：2026-05-31
> 状态：收口中

---

## 当前范围

V1 是本地 MVP，不接入真实 LLM 后端、不做付费、不做账号、不做云同步。

V1 交付内容：

- Flutter 原生 App。
- 本地 SQLite 数据持久化。
- 本地模板生成 AI 点赞和评论。
- 发布文字、图片、视频笔记。
- 首页、发布页、笔记详情、我的、AI 好友、关于、UI 实验室。
- 喜欢、评论喜欢、本地回复、删除本地回复。

---

## 已完成收口项

- 移动端代码已迁移到 `apps/mobile`。
- V1 文档已移除真实 LLM 后端和付费范围。
- 移动端已移除 OpenAI-compatible 直连客户端。
- About 页已说明 V1 不把文字发送到云端 LLM。
- 本地模板互动测试已覆盖。
- 发布页输入框点击外部失焦已修复。
- Flutter 与 iOS 中文本地化已接入。
- 视频发布已提前进入 V1，V1 支持单视频笔记、系统相机拍视频、相册选视频和详情播放。
- 发布页媒体来源已统一为“相机 / 相册 / 取消”，相机入口不再拆成“拍照 / 拍视频”。
