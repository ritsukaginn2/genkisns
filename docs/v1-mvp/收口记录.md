# GenkiSNS V1 收口记录

> 日期：2026-05-31
> 状态：收口中

---

## 当前范围

V1 是本地 MVP，不依赖真实 LLM 后端、不做付费、不做账号、不做实时云同步；iOS 端提供 iCloud 备份/恢复。V1.6 可选接入审核与 LLM 后端，用来做用户/安装实例审核、内容安全审核和服务端 LLM 调用。

V1 交付内容：

- Flutter 原生 App。
- 本地 SQLite 数据持久化。
- iCloud 备份和恢复本机 SQLite 数据库与媒体目录。
- 本地模板生成 AI 点赞和评论。
- 发布文字、图片、视频笔记。
- 首页、发布页、笔记详情、我的、AI 好友、关于、UI 实验室。
- 喜欢、评论喜欢、本地回复、删除本地回复、删除笔记。

---

## 已完成收口项

- 移动端代码已迁移到 `apps/mobile`。
- V1 文档已把真实审核/LLM 后端和付费范围移出 V1 验收。
- 移动端 V1 主流程不依赖 OpenAI-compatible 直连或真实后端；配置 `GENKI_API_BASE` 后，可以通过 `services/llm-proxy` 的 V1.6 后端后台升级真实 LLM 互动。
- About 页已说明 V1 不把文字发送到云端 LLM；文档已补充后续真实 LLM 会经后端审核和代理调用。
- 本地模板互动测试已覆盖。
- 发布页输入框点击外部失焦已修复。
- Flutter 与 iOS 中文本地化已接入。
- 视频发布已提前进入 V1，V1 支持单视频笔记、系统相机拍视频、相册选视频和详情播放。
- 发布页媒体来源已统一为“相机 / 相册 / 取消”，相机入口不再拆成“拍照 / 拍视频”。
- 笔记详情页已补充右上角更多操作，可以删除当前笔记并同步本地数据库。
- 我的页已补充 iCloud 备份入口；iOS 端支持自动备份、手动备份和本机数据缺失时恢复。
