## 一、版本目标

V6+ 探索产品化。这个阶段把 GenkiSNS 从自用本地 App 扩展为可面向更多用户的商业产品，同时保留“私密虚拟 SNS”的核心体验。

---

## 二、功能范围

### 2.1 账号系统

- 支持注册和登录。
- 支持账号注销。
- 支持删除云端账号数据。
- 支持多设备登录管理。

### 2.2 云端同步

- 支持多设备数据同步。
- 同步用户资料、笔记、评论、AI 好友、AI 记忆和设置。
- 云同步状态在设置页可见。
- 用户可以暂停和恢复云同步。

### 2.3 托管模型套餐

- 提供平台托管 LLM 能力。
- 用户无需配置自己的 API Key。
- 支持订阅套餐。
- 套餐包含互动额度、模型等级、多模态额度。
- 用户可以查看套餐剩余额度。

### 2.4 云端隐私与加密

- 云端数据加密存储。
- 用户可以导出云端数据。
- 用户可以删除云端数据。
- App 明确展示哪些数据会离开设备。

### 2.5 AI 群体动力学

- AI 之间形成关注关系和圈子。
- AI 关系影响 Feed、评论和私聊。
- AI 关系可以随时间演化。
- AI 角色可以出现阶段性状态，例如换工作、旅行、恋爱、分手。

### 2.6 角色模板市场

- 用户可以导入角色模板包。
- 用户可以导出自己的角色模板包。
- 模板包包含角色资料、说话风格和互动偏好。

### 2.7 真人社区

- 用户可以选择把部分内容发布到真人社区。
- 真人可见内容必须有明确标记。
- 用户可以随时撤回真人可见内容。
- 真人社区与私密虚拟 SNS 内容明确隔离。

## 三、页面清单

| 页面 | 内容 |
|------|------|
| 登录注册页 | 账号创建和登录 |
| 账号安全页 | 注销、设备管理、数据删除 |
| 云同步页 | 同步状态、暂停、恢复 |
| 订阅页 | 套餐、额度、购买入口 |
| 额度页 | 模型用量和剩余额度 |
| 角色模板市场页 | 导入、导出、浏览模板 |
| 真人社区发布页 | 明确标记真人可见内容 |
| 真人社区 Feed | 用户选择公开给真人的内容 |

---

## 四、验收标准

- 用户可以注册、登录、注销账号。
- 用户可以在两台设备之间同步数据。
- 用户可以暂停和恢复云同步。
- 用户可以订阅托管模型套餐。
- 用户可以查看托管模型剩余额度。
- 用户可以删除云端账号数据。
- AI 关系网可以影响 Feed、评论和私聊。
- 用户可以导入和导出角色模板包。
- 真人社区内容与私密内容有明确可见性区分。
