## 一、版本目标

V5 把模型供应商、模型成本和数据备份交给用户控制。用户可以配置自己的 LLM 供应商，也可以主动备份和恢复本地数据。

---

## 二、功能范围

### 2.1 LLM 供应商配置

- 用户可以添加多组 LLM 供应商。
- 每组供应商包含名称、`base_url`、`api_key`、默认模型。
- 支持 OpenAI 兼容协议。
- 支持连通性测试。
- 支持启用、停用、删除供应商。

### 2.2 模型绑定

- 用户可以设置全局默认模型。
- 用户可以为单个 AI 好友绑定指定模型。
- 用户可以为多模态理解绑定指定模型。
- 用户可以为私聊绑定指定模型。

### 2.3 Key 安全

- API Key 存入 iOS Keychain 或 Android Keystore。
- 日志不记录完整 API Key。
- 数据导出默认不包含 API Key。

### 2.4 用量统计

- 记录每次模型调用的模型名、耗时、token 用量、调用场景。
- 展示每日、每月 token 用量。
- 展示按供应商、模型、功能拆分的用量。

### 2.5 备份恢复

- 支持 iCloud 备份和恢复。
- 支持 Google Drive 备份和恢复。
- 支持手动导出完整数据包。
- 支持从完整数据包导入恢复。

### 2.6 备份内容

- 用户资料。
- AI 好友。
- 笔记、图片、视频。
- 点赞、评论、通知。
- AI 记忆。
- 设置项。

## 三、页面清单

| 页面 | 内容 |
|------|------|
| LLM 供应商列表页 | 查看、添加、启用、删除供应商 |
| LLM 供应商编辑页 | base_url、api_key、模型、连通性测试 |
| 模型绑定页 | 全局、角色、多模态、私聊模型绑定 |
| 用量统计页 | token、耗时、调用场景统计 |
| 备份恢复页 | iCloud、Google Drive、手动导入导出 |
| 备份详情页 | 备份时间、大小、内容摘要 |

---

## 四、关键流程

### 4.1 添加供应商

```text
设置 -> LLM 供应商 -> 添加 -> 填写配置 -> 连通性测试 -> 保存并启用
```

### 4.2 备份恢复

```text
设置 -> 备份恢复 -> 选择备份位置 -> 创建备份或选择备份 -> 恢复数据
```

---

## 五、验收标准

- 用户可以添加并测试至少 1 组自定义 LLM 供应商。
- 用户可以切换全局默认模型。
- 用户可以为单个 AI 好友绑定模型。
- API Key 不以明文写入普通日志或导出文件。
- 用户可以查看 token 用量统计。
- 用户可以创建一次 iCloud 或 Google Drive 备份。
- 用户可以从备份恢复数据。
- 用户可以手动导出和导入完整数据包。
