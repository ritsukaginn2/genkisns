## 一、版本目标

V5 在已有审核与 LLM 后端的基础上，把模型选择、用量统计和数据备份能力开放给用户管理。用户可以选择模型偏好、查看用量和成本提示、主动备份和恢复本地数据。

V5 不要求用户在 App 内填写或保存 LLM 供应商 API Key。真实模型供应商配置仍由后端通过环境变量、云端 secret 或受保护的管理配置维护。

---

## 二、功能范围

### 2.1 模型策略

- 用户可以选择全局模型偏好，例如标准、快速、细腻。
- 用户可以为单个 AI 好友选择模型偏好。
- 用户可以为私聊、多模态理解等场景选择不同模型偏好。
- App 只保存偏好配置，不保存供应商密钥。
- 后端负责把偏好映射到具体供应商、模型和参数。

### 2.2 后端模型配置

- 后端可以维护多组 LLM 供应商配置。
- 每组配置包含名称、`base_url`、模型名和服务端密钥引用。
- 支持 OpenAI-compatible 供应商。
- 支持后端侧连通性测试。
- 支持启用、停用和切换供应商配置。

### 2.3 用量统计

- 记录每次模型调用的模型策略、模型名、耗时、token 用量和调用场景。
- 展示每日、每月 token 用量。
- 展示按功能场景拆分的用量。
- 展示可理解的成本提示，但不要求在 V5 内完成支付闭环。

### 2.4 Key 安全

- App 不保存托管 LLM 供应商 API Key。
- 后端日志不记录完整 API Key。
- 数据导出不包含后端供应商密钥。

### 2.5 备份恢复

- 支持 iCloud 备份和恢复。
- 支持 Google Drive 备份和恢复。
- 支持手动导出完整数据包。
- 支持从完整数据包导入恢复。

### 2.6 备份内容

- 用户资料。
- AI 好友。
- 笔记、图片、视频。
- 点赞、评论、通知。
- AI 记忆。
- 设置项和模型偏好。

## 三、页面清单

| 页面 | 内容 |
|------|------|
| 模型偏好页 | 全局、角色、多模态、私聊模型偏好 |
| 用量统计页 | token、耗时、调用场景统计 |
| 备份恢复页 | iCloud、Google Drive、手动导入导出 |
| 备份详情页 | 备份时间、大小、内容摘要 |

后端供应商配置属于服务端管理能力，不进入移动端用户页面。

---

## 四、关键流程

### 4.1 设置模型偏好

```text
设置 -> 模型偏好 -> 选择全局或角色策略 -> 保存
  -> App 保存偏好
  -> 后续生成请求携带偏好
  -> 后端映射到具体模型并调用
```

### 4.2 备份恢复

```text
设置 -> 备份恢复 -> 选择备份位置 -> 创建备份或选择备份 -> 恢复数据
```

## 五、验收标准

- 用户可以切换全局模型偏好。
- 用户可以为单个 AI 好友设置模型偏好。
- App 不要求用户填写 LLM 供应商 API Key。
- App 不保存托管 LLM 供应商 API Key。
- 用户可以查看 token 用量统计。
- 用户可以创建一次 iCloud 或 Google Drive 备份。
- 用户可以从备份恢复数据。
- 用户可以手动导出和导入完整数据包。
